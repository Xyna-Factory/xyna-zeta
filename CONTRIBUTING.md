## Contributing

[code-of-conduct]: CODE_OF_CONDUCT.md

Please note that this project is released with a [Contributor Code of Conduct][code-of-conduct]. By participating in this project you agree to abide by its terms.

## Guidelines

You also must adhere to the guidelines regarding contributions, described in these articles:
* [Developer Collaboration](https://github.com/Xyna-Factory/xyna-zeta/wiki/Developer-Collaboration) - General collaboration norms for the Zeta project
* [Development Guidelines](https://github.com/Xyna-Factory/xyna-zeta/wiki/Development-Guidelines) - Coding guidelines

## Issues and PRs

If you have suggestions for how this project could be improved or want to report a bug, open an issue.

## Submitting a pull request

1. Fork and clone the repository
1. Create a new branch
1. Make your changes
1. Push your fork and submit a pull request
